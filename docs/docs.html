Aniki Documentation

Welcome to the complete documentation for Aniki (兄貴), an autonomous agent treasury management system built on Sui blockchain. This guide will help you integrate Aniki into your projects and leverage its powerful security and automation features.

OpenClaw Hackathon Entry - This project is part of the OpenClaw Hackathon 2026. Deadline: February 11, 2026.

Installation

Aniki requires Node.js 18+ and access to a Sui network (devnet, testnet, or mainnet).

Prerequisites

# Install Sui CLI
cargo install --locked --git https://github.com/MystenLabs/sui.git --branch devnet sui

# Verify installation
sui --version

Install Aniki

# Clone the repository
git clone https://github.com/your-username/aniki.git
cd aniki

# Install dependencies
npm install

# Build the project
npm run build

Quick Start

Get started with Aniki in under 5 minutes:

1. Environment Setup

# Copy environment template
cp .env.example .env

# Edit .env with your configuration
nano .env

2. Initialize Aniki

import { Aniki } from '@aniki/core';

const aniki = new Aniki({
  network: 'devnet',
  securityLevel: 'high',
  rpcEndpoints: [
    'https://fullnode.devnet.sui.io',
    'https://sui-devnet.nodereal.io'
  ]
});

// Initialize with security configuration
await aniki.initialize({
  coldWallet: '0x123...abc',  // Hardware wallet
  hotWallet: '0x456...def',   // Hot wallet for daily ops
  multiSigThreshold: {
    signers: 3,
    required: 2,
    minAmount: 100000  // SUI units
  }
});

3. Spawn Your First Agent

// Create a secure agent for portfolio management
const agent = await aniki.spawnAgent({
  task: 'Monitor DeFi positions and rebalance portfolio',
  budget: 50000,  // 50K SUI units
  securityLevel: 'standard',
  autoApproval: {
    maxTransaction: 10000,
    dailyLimit: 25000
  }
});

console.log(`Agent spawned: ${agent.id}`);
console.log(`Budget allocated: ${agent.budget} SUI`);

Security Features

Aniki implements multiple layers of security to protect your digital assets:

Air-Gap Wallet Management

Transactions above the configured threshold require cold wallet approval:

// Configure air-gap security
await aniki.configureSecurity({
  coldWallet: {
    address: '0x123...abc',
    type: 'hardware',  // hardware, offline, paper
    approvalMethod: 'qr-code'  // qr-code, usb, manual
  },
  hotWallet: {
    address: '0x456...def',
    maxDaily: 50000,
    maxTransaction: 10000
  }
});

// High-value transaction triggers cold wallet flow
const transaction = await aniki.createTransaction({
  to: '0x789...xyz',
  amount: 150000,  // > coldThreshold
  memo: 'Large DeFi investment'
});

// Returns approval URL for cold wallet signing
console.log(`Cold approval required: ${transaction.approvalUrl}`);

Multi-Signature Support

// Set up multi-sig wallet
await aniki.setupMultiSig({
  signers: [
    '0x123...abc',  // Admin 1
    '0x456...def',  // Admin 2  
    '0x789...xyz'   // Admin 3
  ],
  threshold: 2,  // 2-of-3 signatures required
  minAmount: 25000,  // Minimum for multi-sig
  timelock: 3600     // 1 hour timelock for safety
});

Fraud Detection Engine

// Enable AI-powered fraud detection
await aniki.enableFraudDetection({
  models: ['pattern-analysis', 'velocity-check', 'recipient-scoring'],
  thresholds: {
    suspiciousScore: 0.7,
    blockingScore: 0.9
  },
  monitoring: {
    realTime: true,
    alertWebhook: 'https://alerts.mycompany.com/fraud',
    emailAlerts: ['security@mycompany.com']
  }
});

API Reference

Core Classes

Aniki

Main class for treasury management system.

class Aniki {
  constructor(config: AnikiConfig)
  
  async initialize(securityConfig: SecurityConfig): Promise<void>
  async spawnAgent(agentConfig: AgentConfig): Promise<Agent>
  async getBalance(wallet?: string): Promise<number>
  async createTransaction(tx: TransactionConfig): Promise<Transaction>
  async enableFraudDetection(config: FraudConfig): Promise<void>
}

Agent

Autonomous agent for executing treasury tasks.

class Agent {
  readonly id: string
  readonly budget: number
  readonly securityLevel: SecurityLevel
  
  async execute(task: Task): Promise<TaskResult>
  async requestApproval(amount: number): Promise<boolean>
  async terminate(): Promise<void>
}

Examples

DeFi Portfolio Management

// Create agent for DeFi operations
const defiAgent = await aniki.spawnAgent({
  name: 'DeFi Manager',
  task: 'automated-portfolio-rebalancing',
  budget: 100000,
  parameters: {
    protocols: ['cetus', 'aftermath', 'kriya'],
    riskLevel: 'moderate',
    rebalanceThreshold: 0.05,
    maxSlippage: 0.01
  },
  schedule: {
    type: 'interval',
    frequency: '4h'
  }
});

// Monitor agent performance
setInterval(async () => {
  const status = await defiAgent.getStatus();
  console.log(`Portfolio value: ${status.portfolioValue} SUI`);
  console.log(`24h performance: ${status.performance24h}%`);
}, 60000);

Security Monitoring Agent

// Deploy security monitoring
const securityAgent = await aniki.spawnAgent({
  name: 'Security Monitor',
  task: 'continuous-threat-detection',
  budget: 5000,  // Minimal budget for monitoring
  parameters: {
    scanInterval: 30000,
    threatModels: ['unusual-patterns', 'known-attackers'],
    autoResponse: {
      suspicious: 'alert',
      confirmed: 'lockdown'
    }
  }
});

// Handle security events
securityAgent.on('threat-detected', async (threat) => {
  if (threat.severity === 'high') {
    await aniki.emergencyLockdown();
    await notifySecurityTeam(threat);
  }
});

Troubleshooting

Common Issues

Connection Issues

Error: Failed to connect to Sui RPC endpoint

Solution: Check network configuration and RPC endpoint availability:

# Test RPC connectivity
curl -X POST https://fullnode.devnet.sui.io \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"sui_getLatestSuiSystemState"}'

Gas Estimation Failures

Warning: Transaction gas estimation failed

Solution: Enable gas optimization or increase gas budget:

// Increase gas budget for complex operations
const tx = await aniki.createTransaction({
  to: '0x...',
  amount: 1000,
  gasOptions: {
    budget: 10000000,  // 0.01 SUI
    price: 1000        // Increased gas price
  }
});

Performance Optimization

Transaction Batching

// Enable transaction batching for efficiency
await aniki.configure({
  batching: {
    enabled: true,
    maxBatchSize: 100,
    maxWaitTime: 5000,
    strategy: 'time-or-size'
  }
});
